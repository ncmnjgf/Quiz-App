## ✨ Overview

This is a **modern Quiz Web Application** built using **React JS**.  
It allows users to:

- 🎯 Select quiz criteria (category, difficulty, type)
- 🧠 Answer interactive questions
- 📊 Track performance with a scoreboard
- ⚡ Experience smooth and responsive UI

---

## ⚙️ Features

- 🚀 Dynamic Quiz Generation  
- 🎨 Clean & Responsive UI  
- ⏱️ Timer-based Questions  
- 📈 Score Tracking System  
- 🔄 Smooth Navigation
